# 📊 Personal Investment Portfolio Analysis

A quantitative analysis of my personal investment portfolio using Python, 
covering performance attribution, risk-return analysis, and portfolio optimization.

---

## 🎯 Objective

As a fund manager with 13 years of experience, I built this project to apply 
quantitative methods to my personal portfolio — bridging investment domain expertise 
with data-driven analysis.

## 🔍 Analysis Overview

### 1. Portfolio Overview
- Asset allocation and market value calculation
- Cumulative return analysis by holding
- Risk-return scatter plot

### 2. Benchmark Comparison
- Portfolio vs S&P500 (SPY)
- Active return and tracking error
- Information Ratio

### 3. Risk Analysis
- Annualized return and volatility
- Sharpe Ratio by holding
- Portfolio-level risk metrics

### 4. Portfolio Optimization
- Mean-Variance Optimization (Markowitz)
- Maximum Sharpe Ratio portfolio
- Current vs Optimal weight comparison

---

## 📈 Key Findings

- Portfolio achieved **+31.5% annualized return** (Jun 2025 ~ Apr 2026)
- Current Sharpe Ratio: **1.51** vs S&P500: **1.51**
- Optimal Sharpe Ratio (via MVO): **3.22** — achievable by reweighting toward SCHD, PDBC, TSLA
- KODEX REITs showed lowest risk-adjusted return (Sharpe: 0.35) in this period
